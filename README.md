## Installing

Each `.skill` file can be uploaded as a skill (via the Skills API, claude.ai capabilities/skills settings, or dropped into a Claude Code skills directory). Install only the ones you want — they're independent.

## A note on scope

These are coaching skills built from general domain expertise, themed to match each course. They deliberately do not copy WebVeda's lesson content. Several (tax, investing, the India-specific ones) include reminders to verify current figures and consult a professional, since rules change and Claude isn't a licensed advisor.
